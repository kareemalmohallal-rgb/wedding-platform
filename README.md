# Event Contractors Platform 🎉

A web application for managing event contractors, services, and user ratings.

## 🚀 Features
- Manage contractors and services  
- Add and view ratings  
- Responsive web interface  
- REST API backend

## 🛠️ Built With
- ASP.NET Core  
- REST APIs  
- SQL Server  
- HTML, CSS, JavaScript  

## 📌 Purpose
This project was built as a personal learning project to practice full stack web development and real-world system design.
