# Event Contractors Platform 🎉

A responsive web application for managing event contractors and services. The project allows users to browse contractors, view services, and interact with a clean and simple user interface.

---

## 🚀 Features
- Display event contractors and their services  
- View detailed contractor information  
- User ratings interface (UI only)  
- Fully responsive design for all devices  
- Clean and user-friendly front-end layout  

---

## 🛠️ Built With
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## 📌 Purpose
This project was built as a personal learning project to improve front-end development skills and practice building real-world web interfaces using core web technologies.
